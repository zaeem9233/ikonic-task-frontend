# ikonic-task

## Project setup

### Clone the Repository

```
git clone https://github.com/zaeem9233/ikonic-task-frontend
```

### Change Directory
```
cd ikonic-task-backend
```

### Get Dependencies
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
